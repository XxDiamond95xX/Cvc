# Formation Technicien CVC

Application React + Vite, responsive, optimisée pour mobile et prête pour Vercel.

## Lancer en local

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Déployer sur Vercel

- Importer le dossier dans Vercel
- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
